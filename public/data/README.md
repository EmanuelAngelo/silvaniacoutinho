# Pasta de imagens (`public/data`)

Coloque aqui as imagens do site para ficar organizado.

## Estrutura sugerida

- `public/data/hero/`
  - Imagens de fundo do topo (hero) do site.
  - Arquivos de exemplo: `clinic-1.jpeg`, `clinic-2.jpeg`, `clinic-3.jpeg`

- `public/data/areas/`
  - `clareamento/` (ex.: `1.jpeg`, `2.jpeg`, `3.jpeg`)
  - `ortodontia/`
  - `implantes/`
  - `estetica-do-sorriso/`
  - `prevencao/`
  - `clinica-geral/`

## Como trocar

Basta substituir os arquivos mantendo os mesmos nomes, ou adicionar novos e atualizar as listas em `src/content/media.ts`.
